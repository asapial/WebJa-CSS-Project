CSS clip-path Property Values

clip-path: none

This is fully visible content.

clip-path: url(#myClipPath)

This is clipped using an SVG path.

clip-path: inset(10px 20px 30px 40px)

This is clipped content.

clip-path: circle(50% at 50% 50%)

This is circular clipped content.

clip-path: ellipse(50% 30% at 50% 50%)

This is elliptical clipped content.

clip-path: polygon(50% 0%, 0% 100%, 100% 100%)

This is polygonal clipped content.

clip-path: path('M10 10 H 90 V 90 H 10 L 10 10')

This is path-based clipped content.

clip-path: inherit

This inherits clipping from its parent.
